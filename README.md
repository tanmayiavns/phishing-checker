# Phishing URL Detector 🔍

## 📌 Overview
This project detects phishing URLs using heuristic-based analysis.

## ⚙️ Features
- URL structure analysis
- Keyword detection
- Protocol checking (HTTP/HTTPS)
- IP-based URL detection
- Risk scoring system

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS

## 🚀 How to Run
1. Install Flask
2. Run app.py
3. Open http://127.0.0.1:5000

## 🧠 Concept
This system uses rule-based detection instead of blacklist databases, making it adaptable to new phishing patterns.

## 🔮 Future Scope
- Machine learning integration
- Browser extension
- Real-time API scanning
